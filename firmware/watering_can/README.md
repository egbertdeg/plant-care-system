# Watering Can Firmware

ESP32-S3 firmware for the smart watering can.

## Hardware
- Adafruit ESP32-S3 Feather (#5477)
- LSM6DS3TR-C IMU (tilt detection)
- MPRLS pressure sensor (water volume)
- 128×64 OLED display
- 400mAh LiPo battery
