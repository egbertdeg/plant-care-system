# Plant Care System

Smart IoT system for monitoring plants and intelligent watering.

## Hardware
- ESP32-S3 sensor pods (moisture, temp, humidity, light)
- ESP32-S3 smart watering can (tilt detection, volume measurement)

## Components
- `/firmware` - ESP32 embedded code
- `/backend` - Cloud API (FastAPI + PostgreSQL) *(planned)*
- `/ml` - ML models for watering prediction *(planned)*
- `/mobile` - Flutter mobile app *(planned)*
- `/docs` - Documentation
